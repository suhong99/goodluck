## 파이어베이스 data 구조
```markdown
USERS
└── userId (Document)
├── ENFORCEMENT (Sub-Collection)
│ └── id (Document)
│ ├── date(fieldA): TimeStamp
│ ├── percent(fieldB): number
│ └── status(fieldC): string
└── SHIBA (Sub-Collection)
└── id (Document)
│ ├── date(fieldA): TimeStamp
└── type(fieldB): string
```
